# PolkaStream
PolkaStream - The first native stablecoin streaming payments protocol on Polkadot Hub EVM. Supports Cliff delays, real-time withdrawals, pause/resume, multi-token, and atomic XCM cross-chain notifications. Built for Polkadot Solidity Hackathon 2026 (Track 1 - Stablecoin-enabled dApps + Polkadot Native Integration).
